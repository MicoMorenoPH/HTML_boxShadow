1. Explaination - box shadow properies [box-shadow: horizontal_distance vertical_distance blur_radius color] The box-shadow property renders a shadow for the given object. A. HORIZONTAL_DISTANCE Horizontal distance of the shadow relative to actual text. Negative horizontal_distance moves the shadow towards left. Positive horizontal_distance moves the shadow towards right. B.VERTICAL_DISTANCE Vertical distance of the shadow relative to actual text. Negative vertical_distance moves the shadow up. Positive vertical_distance moves the shadow down. C.BLUR_DISTANCE How blurry the shadow will appear. D.COLOR Color of the shadow. It can be specified along with an alpha value for transparency (as shown in the example).

The box-shadow Property

The box-shadow property defines the shadow of an element:

box-shadow: 5px 10px:

A div element with a shadow. The first value is the horizontal offset and the second value is the vertical offset. The shadow color will be inherited from the text color.

box-shadow: 5px 10px #888888:

You can also define the color of the shadow. Here the shadow color is grey.

box-shadow: 5px 10px red:

A red shadow.
